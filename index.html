PURE COMPONENETS

React component is considered pure if it renders the same output for the same state and props.
For this type of class component, React provides the PureComponent base class. Class components that
extend the React.PureComponent class are treated as pure components.>

        import React, { PureComponent } from "react";

        export default class PureClassComponent extends PureComponent {
          constructor() {
            super();
            this.state = {
              count: 0
            };
          }
        
          handleClick = () => {
            this.setState((prevstate) => ({
              count: prevstate.count + 1
            }));
          };
        
          render() {
            return (
              <>
                
{this.state.count}

          }
